## For users

- **API keys:** create and revoke keys from [Dashboard → API keys](/dashboard/api-keys). The full secret is shown only once after creation; the database stores `sha256_v1` of the plaintext plus a short prefix for identification.

- Apply `supabase/migrations/20250511153000_api_keys.sql` (see `docs/SUPABASE.md` in the repo).
