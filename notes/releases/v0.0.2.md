# DocRail 0.0.2

## For users

- **Sign up and sign in** with email and password (Supabase Auth).
- **Organization onboarding** — create your first organization after registering.
- **Dashboard** shell for signed-in users who completed onboarding.
- **Sign out** from the site footer when authenticated.

## Setup

Configure Supabase environment variables and run the SQL migration described in the repository docs before using auth features.
