# DocRail 0.0.5

## For users

- **Templates** at [Dashboard → Templates](/dashboard/templates): create **Blank** or **Letter** layouts per organization; letter fields are validated server-side (`subject`, `body` length limits).
- **PDF preview:** letter templates include an in-dashboard preview powered by `@react-pdf/renderer` (`PDFViewer` loads only on the client).
- Edit template names/content, toggle preview, or delete templates (with confirmation).

## For operators

- Apply `supabase/migrations/20250511160000_templates.sql`. Install JS deps (`npm ci`) — this release adds **`@react-pdf/renderer`** and enables `transpilePackages` for it in `next.config.ts`.
